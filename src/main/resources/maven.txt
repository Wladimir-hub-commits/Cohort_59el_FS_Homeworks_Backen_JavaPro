clean - удаляет результат предыдущей сборки типично clean package

- validate - быстрая проверка, что проект валидный,есть валидный pom и базовая структура

compile - компиляция src/main/java --> target/classes быстрая проверка компиляции кода

test - запускает unit-тесты типично перед package
mvn clean package -DskipTests команда для пропрыгивания unit-tests

- package - собирает наш артефакт пакета и кладёт его в target

- verify - проверка качества сборки, валидация артефактов, плагины контроля

- install - калдёт артефакт проекта в локальный репозиторий

- site - документация и отчёты

deploy - публикует наш артеакт на удалённом репозитории, делает релизную версию. Альтернатива: Jenkins, Tekton
mvn dependency:tree вывод на экран дерева зависимостей
